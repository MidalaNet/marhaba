# Marhaba
## Marhaba Core
### Requirements

* libcurl
* libsqlite3
* libmxml
* libtidy
### Build from source
Change to 'src' and type:

      make
      make install
      make clean
## Mar-haba Bot
Change to 'bot' and type:

    ./bot.pl
Marhaba Bot requires two additional Perl modules, provided by CPAN:

* LWP
* XML::Feed
